# Immersive Focus Companion (MVP)

可运行的 Next.js + TypeScript + TailwindCSS 工程，包含：
- 番茄钟（开始/暂停/快进/立即结束）
- 音乐与环境混音（含本地文件导入）
- 虚拟人物 A 陪伴
- 笔记、习惯、待办页面
- API 路由（focus/notes/habits/todos/track）

## 运行
```bash
npm install
npm run dev
```
访问 `http://localhost:3000`

## 关键文件
- `app/page.tsx`
- `components/*`
- `app/api/*`
- `lib/reward.ts`
- `IMMERSIVE_FOCUS_MVP_PLAN.md`
