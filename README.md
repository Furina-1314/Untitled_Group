# Immersive Focus Companion

完整功能工程：
- 番茄钟倒计时（开始/暂停/恢复/快进/立即结束结算XP）
- 音乐与环境（导入、播放、下一首、删除、音量控制）
- 虚拟人物交互（情境语气变化、语音播报）
- 笔记、日历日记、习惯追踪、待办管理（增删改查）

## Visual Studio 2022 运行方式（Windows）
1. 安装 Node.js 20+
2. VS2022 -> File -> Open -> Folder，打开本项目根目录
3. 在终端执行：
   ```bash
   npm install
   npm run dev
   ```
4. 访问 `http://localhost:3000`

## 命令
```bash
npm run dev
npm run build
npm run start
npm run typecheck
```
